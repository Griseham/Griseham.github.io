AWS TwinMaker Project

As part of a team, our group created a digital version of our university’s library, the FAU S.E. Wimberly Library, using AWS services such as AWS TwinMaker, Timestream, Grafana and more. This project was aimed to help students and faculty glean information about the library from just a website. The website gave information such as occupancy, computer availability, temperature, class demographics and more. We presented this project for our Summer 2022 senior showcase and came out as one of the 3 winning projects
