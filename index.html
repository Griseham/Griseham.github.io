About me

My Introduction

I’ve been in school all my life. from taking college courses (Honors and AP) in high school, to living on-campus at my university, taking classes every coming semester until I graduate. Though I’ve learned something new from each class, I realized that the time and effort I spent on some of them wouldn’t help me in ways that I expected. I used to think for every hard class I took, I’d become smarter, and as long as my GPA went up, I wasn’t really wasting my time. I worked so hard molding the life of a person I wanted to be, that I forgot that life can only be lived in the present, creating works that you are excited about and learning things that you choose to invest the time into. Now I’ve stopped worrying about the future and chose to work within the realm of my immediate vision. I want to learn difficult coding languages to make meaningful projects. I’m excited to spend all nighters creating apps that serve the public, instead of serving a test. I must take small steps to improve myself now, rather than be defined by a past me, long gone to time.

Academic Projects

AWS TwinMaker Project

As part of a team, our group created a digital version of our university’s library, the FAU S.E. Wimberly Library, using AWS services such as AWS TwinMaker, Timestream, Grafana and more. This project was aimed to help students and faculty glean information about the library from just a website. The website gave information such as occupancy, computer availability, temperature, class demographics and more. We presented this project for our Summer 2022 senior showcase and came out as one of the 3 winning projects

Clicking the demo button will download a video file of our group’s presentation.

Sorting Visualizer

This sorting visualizer project is adapted from a sorting algorithm tutorial on YouTube. This visualizer creates an array populated by random values, to be then used to create a set of rectangles. Using merge sort, the visualizer organizes the rectangles from the lowest value to the highest.
